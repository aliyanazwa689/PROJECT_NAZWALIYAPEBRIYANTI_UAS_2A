about

Website ini saya buat sebagai media untuk memperkenalkan diri serta berbagi informasi mengenai profil, pendidikan, hobi, dan berbagai hal yang saya minati. Selain itu, website ini juga menjadi sarana bagi saya untuk belajar, berlatih, dan mengembangkan kemampuan di bidang teknologi, khususnya dalam pengembangan dan pembuatan website.

Melalui website ini, saya ingin mendokumentasikan perjalanan belajar, pengalaman, serta berbagai proyek yang pernah saya kerjakan. Saya juga berharap website ini dapat menjadi wadah untuk berbagi informasi, inspirasi, dan wawasan yang bermanfaat bagi para pengunjung.

Dengan adanya website ini, saya berharap pengunjung dapat mengenal saya lebih dekat, memahami minat dan bidang yang saya tekuni, serta menjalin komunikasi dan kolaborasi yang positif di masa mendatang. Terima kasih telah berkunjung dan meluangkan waktu untuk menjelajahi website ini. Semoga informasi yang saya bagikan dapat memberikan manfaat dan menambah wawasan bagi para pengunjung.

profile

Saya berasal dari Lampung dan sedang menempuh pendidikan di Universitas Muhammadiyah Kotabumi pada Program Studi Sistem dan Teknologi Informasi. Saya memiliki beberapa hobi, di antaranya jalan-jalan, memasak, dan membaca novel, serta warna favorit saya adalah lilac.
